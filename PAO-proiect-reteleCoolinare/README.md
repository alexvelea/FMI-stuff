# reteteCoolinare
PAO 2018
```
Soft retete culinare
 Sistem de autentificare cu nume de utilizator si parola;
 Client pentru administrarea retetelor – administatorii pot sa faca operatii
de tip CRUD (create, read, update, delete) pe retete: ingredient (o reteta
poate sa contina mai multe ingrediente,in cantitati diferite), timp de
preparare, nivel de complexitate (usor, mediu, dificil), descriere reteta,
cost aproximativ ingrediente.
 Client pentru utilizatorii normali – clientii pot vedea retetele existente in
baza de date si le pot filtra in functie de o lista de ingrediente introdusa, in
functie de complexitate si pret;
```
